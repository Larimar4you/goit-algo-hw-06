# DFS та BFS

# Граф

A — B — C — D — E — F
(заданий через список ребер)

## Виконані дії

- Реалізовано DFS через стек
- Реалізовано BFS через deque
- Побудовано шляхи від стартового вузла 'A'

## Результати

DFS path:['A', 'B', 'C', 'E', 'D', 'F']
BFS path: ['A', 'B', 'C', 'D', 'E', 'F']

# Пояснення

- DFS йде вглиб: обирає один шлях і досліджує його до кінця
- BFS йде вшир: відвідує всі сусіди поточного вузла перед переходом на наступний рівень
- Різниця в шляхах обумовлена методикою обходу графа
